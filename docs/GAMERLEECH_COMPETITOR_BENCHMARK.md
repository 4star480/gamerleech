# GamerLeech — Competitor Benchmark (Phase 2)

## Sites analyzed

| Site | Strengths | Patterns to adopt | Avoid |
|------|-----------|-------------------|-------|
| [Hooked.wtf](https://hooked.wtf/) | Status transparency, “checkout → in-game” timeline, Discord support CTA | Step-by-step post-purchase flow, instant delivery messaging | Over-long copy |
| [ZhexCheats](https://zhexcheats.com/en) | Product status page, subscription pause on downtime, dashboard keys | “Undetected” badges, live status, minimal catalogue trust | Aggressive fear marketing |
| Stripe checkout guides | Order summary sticky, explicit CTA copy, inline validation | Single-column mobile checkout, large pay button | Enterprise complexity |
| CoinGate gaming | Crypto as first-class payment | Crypto icons + clear network labels | White-label overhead |

## Layout patterns (niche standard)

1. **Hero:** One headline + sub + primary CTA to catalogue + trust row (instant / secure / support)
2. **Shop:** Category tiles → product grid; status badge per SKU where possible
3. **Mobile:** Bottom nav (Home / Shop / Cart / Support) — industry norm on mobile-first cheat stores
4. **Checkout:** Order summary first on mobile; crypto selector as large tiles; email capture before pay
5. **Trust:** FAQ + refund policy linked (not `#`); email + Discord visible

## GamerLeech positioning (v2)

**Direction:** Premium dark + electric green — cleaner than generic “hacker green on black,” closer to Hooked’s clarity without copying.

**Tone:** Confident, short sentences. Lead with delivery speed and support — not fear.

## Mood board summary

- **Background:** `#060a06` near-black with subtle green radial glow
- **Accent:** `#2ef47a` (brighter than old `#28e06d`)
- **Type:** Urbanist 600/800 — display weight on headings only
- **Cards:** 14px radius, 1px border `#1a2a1a`, hover glow
- **Motion:** 150–300ms ease-out; respect `prefers-reduced-motion`
